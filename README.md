# Loan-dataset-using-isolationForest-for-outlier

The provided script builds a complete scikit-learn machine learning pipeline that handles preprocessing, outlier detection, class imbalance, and model evaluation. It uses an IsolationForest to identify and remove the top 1% of extreme anomalies from the training data before fitting the main model. To handle class imbalance, a SMOTEENN sampler combines synthetic data generation with editing links to balance the target classes smoothly. A RandomForestClassifier is then trained on this clean, balanced data and evaluated on a holdout test set to simulate real-world production performance. Finally, the code extracts post-processed column names directly from the ColumnTransformer to generate an indexed bar plot of feature importances.
